# Emotion Detection Application

## Overview
This repository contains the final project for the **IBM AI Application Development** course on Coursera. The project is an Emotion Detection Web Application that analyzes text and determines the underlying emotions using the Watson NLP library.

## Project Details
* **Project Name:** Emotion Detection
* **Author:** TLayol (hoặc điền tên thật của bạn vào đây)

## Description
The application utilizes a Flask web server to provide a user interface where users can input text. The text is then processed to extract a score for various emotions including:
- Anger
- Disgust
- Fear
- Joy
- Sadness

The system will then return these scores along with the **dominant emotion**. It also includes robust error handling to manage empty or invalid inputs.

## Technologies Used
* Python
* Flask
* IBM Watson NLP Service
* HTML/CSS (Front-end)
